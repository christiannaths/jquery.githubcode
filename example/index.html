jQuery GithubCode

Show code from any public Github repository.

GithubCode is a plugin for jQuery 1.7.2 or higher which will fetch code from public Github repositories for display on your website. Useful for technical blogging or tutorial writing, it allows you to maintain and update your Github repo as part of your publishing process.
